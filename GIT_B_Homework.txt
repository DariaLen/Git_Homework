1. Как скопировать на свой компьютер репозиторий из github.com, что должно быть сделано предварительно?

	Для начала нужно взять ссылку ssh на репозиторий из GitHub 
	В Git Bash ввести команду :
	git clone ‘ссылка’

2.Как создать git репозиторий из локального каталога на своем компьютере. Как его отправить на github.com?

	Нужно в редакторе nano создать файл ( nano name.txt) написать  в него код . Далее нужно отправить его на GitHub ( git add fileName) ,
 	далее ввести команду git add и затем git commit -m «комментарий» и затем команда git push ( но прежде перепроверить все ли сделано без ошибок ) 


3.Как получить на локальный компьютер изменения с github.com

	git pull ( подтянуть обновления)

4.Какую информацию нам дают команда :

	git status  ----> говорит где я , на какой ветке , покажет не отслеживаемые гитом файлы , они будут  подсвечиваться красным цветом 
	git log     ----> git log покажет список всех коммитов от старых к новым , виден уникальный код коммита 
	git remote -v ---> проверит , связан ли репозиторий с гитхаб
